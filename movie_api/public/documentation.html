Documentation

My Flix is a web application that provides users with the ability to search and save different information about movies! They will be able to sign-up, update their personal information as well as creat a list of their own favourite movies!

Description URL HTTP method Request body data format
Return a list of All movies to the user /movies GET A JSON object holding data about all movies.
Return data (description, genre, director) about a single movie by title to the userâ€¦ /movies/:title GET A JSON object holding data about a single movie, containing an ID, title, and some other properties.
Return data about a genres (description) by movie title. movies/genre/:name GET A JSON object holding data about a genre (description) by name or title.
Return data about a director by name /movies/director/:name GET A JSON object holding data about a single director, containing name, birth year, death year, movies.
Allow new users to register /users POST A JSON object holding data about the new user to register/add
Allow users to update their user info (username, password, email, date of birth) /users/:name PUT A JSON object holding data about the current user to update which gives a text message indicating whether the user's account has succesfully changed
Allow users to add a movie to their list of favorites /users/:name/movies/:movie_id POST A JSON object holding data about the movie to add to the user's list of favorite movies which gives a text message indicating whether the user's favorite movie was succesfully added to their list.
Allow users to remove a movie to their list of favorites /users/:name/movies/:movie_id DELETE A text message indicating whether the movie was succesfully removed from the users' favorite-list.
Allow existing users to deregister (remove a user by name or id) /users/:name DELETE A text message indicating whether the user's account was succesfully removed.
